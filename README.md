# Thiago Pereira affiliate partner profile

Static GitHub Pages profile for affiliate partner applications.

Public site:

https://thiago-a-pereira.github.io/

The page presents an independent, conversion-focused affiliate operation with a compliance-first approach. It is designed for partner approvals and does not claim fake results, fake testimonials, unauthorized brand affiliation, or guaranteed outcomes.

Files:

- `index.html`: partner profile page.
- `privacy.html`: basic privacy policy.
- `contact.html`: contact note for partner applications.
- `assets/site.css`: responsive styling without external dependencies.
- `assets/tracking.js`: local outbound-click tracking stub.
